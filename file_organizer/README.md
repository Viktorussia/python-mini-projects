# File Organizer

Sorts files in a выбранной папке intro subfolders by extension:
Images, Documents, Archives, Audio, Video, Other.

## Run
'''bash
python organizer.py
'''

### Example

Input: 
D:\Downloads

Output:
Готово. Файлы отсортированы в: D:\Downloads

## Важно по безопасности

Перед использованием рекомендуется тестировать скрипт на копии папки, 
а не на основной директории (например, 'Downloads').

Пример:
1. Создайте тестовую папку 'D:\test_files'
2. Скопируйте туда несколько файлов
3. Запустите скрипт и укажите путь к тестовой папке

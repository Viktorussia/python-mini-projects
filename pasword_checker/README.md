## Password Checker

### Описание
Скрипт проверяет надежность пароля по базовым правилам безопасности.

### Проверяет
- длина минимум 8 символов
- наличие строчных и заглавных букв
- наличие цифр
- наличие спецсимволов
- отсутствие пробелов

### Запуск
'''bash
python password_checker.py
'''
### Пример

Ввод: qwerty

Вывод: Пароль слабый. Исправь:
1. Минимум 8 символов 
2. Добавь заглавную букву (A-Z)
3. Добавь цифру (0-9)
4. Добавь спецсимвол (!@#$...)

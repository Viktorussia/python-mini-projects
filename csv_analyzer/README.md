## CSV Analyzer

### Описание
Скрипт читает CSV-файл и расчитывает:
- rколичество записей
- общую сумму
- среднее значение 

### Запуск
'''bash
python csv_analyzer.py

### Пример
Файл sample.csv:

name,amount
Order1,1200
Order2,800
Order3,1500
Order4,500

Вывод:
Количество записей: 4
Сумма: 4000
Среднее значение: 1000.00
